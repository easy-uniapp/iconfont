## Install

```sh
# yarn
yarn add @easy-uniapp/iconfont --dev

# npm
npm install @easy-uniapp/iconfont --save-dev
```

## Usage

Generate iconfont.scss (src/styles/iconfont.scss)

```sh
# yarn
yarn iconfont <font class url>

# npm
npx iconfont <font class url>
```

Use iconfont.scss

```scss
@import '~@/styles/iconfont';
```
